Python如何操作文件

2019-02-27

好记性比如烂笔头，计算机也是如此。

当我们利用计算机计算得到大量数据之后，如果不能将这些数据保存起来，肯定是一件十分遗憾的事情，那么我们今天来学习一下如何使用Python来保存数据。

Python的open()函数

在Python中使用open()函数，可以打开打开一个新的文件，或者创建一个新的文件

open(文件名，访问模式)

访问模式 说明
r 以只读的方式打开文件。文件指针默认在文件文件的开头。
w 打开一个新的文件只用于写入，如果文件存在则将其覆盖，如果文件不存在，则创建新的文件。
a 打开一个文件用于追加，如果文件存在，文件指针则在文件末尾。新的内容加入到已有内容之后，如果文件不存在，创建新的文件。
rb 以二进制格式打开一个文件用于只读，文件指针放在文件开头。
wb 以二进制格式打开一个文件用于写入，如果该文件夹存在则覆盖。如果文件不存在，创建一个新的文件。
ab 以二进制格式打开一个文件用于追加，如果文件已存在，文件指针将会放在文件结尾。
r+ 打开一个文件用于读写，文件指针放在文件开头。
w+ 打开一个文件用于读写。如果1该文件已存在则将其覆盖，如果文件不存在，创建新文件。

关闭文件

打开一个文件，对文件的内容进行操作之后，需要对文件进行关闭，使用到close()函数。
